# Sales Analysis Dashboard 📊

## Project Overview
This is an end-to-end sales analysis project using Python, SQL, and Power BI.

## Tools Used
- Python (Data Creation & Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard)

##  Dashboard Features
- Sales by Category
- Top 10 Cities by Sales
- Monthly Sales Trend
- KPI Cards (Total Sales, Orders, Quantity)
- Interactive Filters (Slicer)

##  Key Insights
- Top Sales City: Bengaluru
- Best Selling Category: Kurta
- Peak Sales Month: April
